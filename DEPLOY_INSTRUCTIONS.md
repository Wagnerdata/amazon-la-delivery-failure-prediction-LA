# 🚀 Guia de Entrega: Link do Dashboard (Grátis)

Siga estes passos para gerar o link oficial do seu projeto e compartilhar com sua liderança ou no Correlation One.

## Passo 1: Subir o Código para o GitHub
O Streamlit Cloud precisa que seu código esteja no GitHub para gerar o link.

1. Acesse [github.com](https://github.com/) e crie uma conta (se não tiver).
2. Crie um novo repositório chamado `amazon-delivery-prediction`.
3. Siga as instruções do GitHub para subir os arquivos desta pasta para lá.
   - *Dica: Você pode simplesmente arrastar e soltar os arquivos no site do GitHub se preferir.*

## Passo 2: Gerar o Link no Streamlit Cloud
1. Acesse [share.streamlit.io](https://share.streamlit.io/).
2. Clique em **"Connect GitHub"**.
3. Clique em **"Deploy an app"**.
4. Configure as opções assim:
   - **Repository:** `seu-usuario/amazon-delivery-prediction`
   - **Branch:** `main` (ou `master`)
   - **Main file path:** `dashboard/dashboard.py` (Isso é muito importante!)
5. Clique em **"Deploy!"**.

## Passo 3: Compartilhar o Link
Em 1 ou 2 minutos, o site estará no ar. O link será algo como:
`https://sua-url-customizada.streamlit.app/`
